
# Metadata Forensics Suite BETA

Advanced browser-based forensic metadata analyzer.

## Features

- EXIF parser
- PNG chunk parser
- eXIf detection
- XMP text extraction
- C2PA detection
- JUMBF detection
- AI signature detection
- OpenAI detection
- Midjourney detection
- Stable Diffusion detection
- Flux detection
- Adobe Firefly detection
- SHA1
- SHA256
- MD5
- Raw header analysis
- Multi-file support
- ZIP export
- Metadata cleaner
- Offline processing
- GitHub Pages ready

## Deploy

Upload to GitHub Pages and open index.html.
